Icons

Icons should be descriptive, simple, readable, and consistent. Use them to communicate an action, status, or help separate objects in a list. In other words, icons can represent objects, tools, actions, or desired results.

Available Icons

All icons are part of a category based on primary purpose. Some generic icons will by used outside the named purposed due to utility.

To learn how to use the icons see the Icons documentation.
