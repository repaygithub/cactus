Base Grid

The base grid is the reference from which the rest of the layout structures are built. It defines the starting point of the dimensions, as well as the paddings and margins of the elements of the interface.

The grid is built from a base 8 pixel module so that both the sizes of the elements and the spaces between them will always be multiples of 8: 16, 24, 32, 40, 48.

There may be situations, however where a value smaller than 8px is required; the 4px unit is available for these cases.

This grid enables the creation of a standard interface by defining the abstract composition of an page and how individual components associate within an interface.

The main bar, footer, and action bar use the 8pt grid for placement.

Our product is composed of
1) Main Bar (Quickline Bar)
2) Action Bar
3) Side panel
4) Content
5) Footer


Column
The columns are fluid, having a responsive width which conforms to the size of their parent container.
Margin
Margins are the negative space between the edge of the format and the outer edge of the content.
Gutter
Grid columns are seperated by areas of white space referred to as "gutters". Gutters improve legibility by providing negative space between page elements.

Break-points

The different column sizes automatically change dimensions if they reach a breakpoint.
