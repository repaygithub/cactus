Available Icons

The icon library allows icons to be imported in two different manners.

From the root by name:

import * as React from 'react'
import { ActionsAdd, ActionsDelete } from '@repay/cactus-icons'

export default () => (
  <React.Fragment>
    <ActionsAdd />
    <ActionsDelete />
  </React.Fragment>
)

Individually by file path:

import * as React from 'react'
import ActionsAdd from '@repay/cactus-icons/i/actions-add'
import ActionsDelete from '@repay/cactus-icons/i/actions-delete'

export default () => (
  <React.Fragment>
    <ActionsAdd />
    <ActionsDelete />
  </React.Fragment>
)
