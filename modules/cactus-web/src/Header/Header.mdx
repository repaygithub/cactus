# Header

The `Header` component allows you to set a heading `Title` with aditional elements: `Breadcrumbs` or `Back buttons` and other different items.

## Basic usage

- The `Header` component has `Header.Title`, `Header.Item` and `Header.BreadcrumbRow` sub-components.
- The `Header.Title` is the most important sub-component, you must use it with the `Header` component.
- The `Header.BreadcrumbRow` allows you to wrap a `Breadcrumb` component or a `Go Back` button.
- The `Header.Item` allows you to wrap additional items that will be right-aligned (in desktop views).
- The `Header` component uses its sub-components to arrange the elements you want to insert inside the header. If you pass a `children` without being wrapped into any subcomponent, that children won't be rendered.

```jsx
import * as React from 'react'
import { Header } from '@repay/cactus-web'
...
<Header>
  <Header.Title>Heading Title</Header.Title>
</Header>
```

### Try It out

export const code = `
    <Header>
      <Header.BreadcrumbRow>
        <Link to="https://repaygithub.github.io/cactus/">Go back!</Link>
      </Header.BreadcrumbRow>
      <Header.Title>Heading Title</Header.Title>
      <Header.Item>
        <Button variant="action">
          <Icon /> Add new configuration
        </Button>
      </Header.Item>
      <Header.Item>Some text because I like crowded UIs</Header.Item>
    </Header>
`
