## Basic usage

The `Link` component can be used by passing the url in the `href` prop, and providing the desired text as the children.

```jsx
import React from 'react'
import { Link } from '@repay/cactus-web'

const MyComponent = props => {
  // normal component things here
  return (
    <div>
      <Link href="https://google.com">Go to google</Link>
      // rest of component
    </div>
  )
}
```
