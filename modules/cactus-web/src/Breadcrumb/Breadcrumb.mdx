# Breadcrumb

The `Breadcrumb` comopnent allows you to render a navigation of your previusly visited sites, including the current one

### Try it out

export const code = `<Breadcrumb>
  <Breadcrumb.Item linkTo="/">Account</Breadcrumb.Item>
  <Breadcrumb.Item linkTo="/" active>Make a Payment</Breadcrumb.Item>
</Breadcrumb>`

<LiveProvider code={code} scope={{ Breadcrumb }}>
  <LiveEditor style={livePreviewStyle} />
  <LiveError />
  <LivePreview />
</LiveProvider>

## Best practices

The Breadcrumb component consists of two parts: The `Breadcrumb` which works as a wrapper for the navigation links.
And a `Breadcrumb.Item` which represents a single Breadrumb or link. linkTo should be passed as a prop,
and if it's the last navigation Breadcrumb, an active prop should be passed as well.
